Pantheon Lighting System

Pantheon in a Nuke plugin that allows productions to add volumetric lighting to 2D animation. The plugin was originally developed by Noah Catan for The Popes Dog. After collaborating with Noah, I added two nodes to the toolset that improve artist efficiency.

Framehold - This node evaluates frame sequences to determine if frames are being held for more than one frame. Prior to it's introduction, artist would have to manually keyframe the default framehold node which would take ~30 minutes - 1 hour for every shot. The process has now been simplified to take less than 10 seconds.

MatteLight - This node compliments the manual rotoscoping of lighting shapes by offering a procedural approach. There are also controls to override the procedural lighting with manual rotoscoping if it creates odd behavior.
